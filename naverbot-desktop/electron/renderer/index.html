블로그와 카페 자동화에서 공통으로 사용할 네이버 계정을 관리합니다.

저장된 계정

저장된 계정이 없습니다. 아래에서 계정을 추가하세요.

계정 추가/변경

버튼을 누르면 네이버 로그인 창이 열립니다.
직접 로그인하면 세션이 저장되어 이후 자동으로 로그인됩니다.
비밀번호가 바뀌었거나 다른 계정 로그인 기록이 남아 있으면 초기화 후 다시 로그인하세요.

로그인이 안 되나요?

1단계: 캡차(보안문자) 인증
네이버가 봇을 의심하면 캡차가 나타납니다.
"네이버 로그인" 버튼을 누르면 브라우저가 열리니,
캡차를 직접 풀고 로그인을 완료해주세요.

2단계: 2차 인증 (OTP/전화)
2차 인증이 설정되어 있다면 인증을 완료해주세요.
로그인이 끝나면 자동으로 세션이 저장됩니다.

3단계: 그래도 안 된다면
1. 일반 브라우저(Chrome)에서 먼저 네이버에 로그인하세요.
2. 로그인 상태에서 이 앱의 "네이버 로그인" 버튼을 다시 눌러주세요.
3. 네이버 앱에서 "새 기기 로그인 허용"을 확인해주세요.

4단계: 비밀번호 변경/다른 계정 기록
저장된 로그인 초기화 버튼으로 기존 세션과 저장 비밀번호를 지운 뒤 새 비밀번호로 다시 로그인하세요.

세션은 한 번만 저장하면 이후 자동으로 로그인됩니다.
