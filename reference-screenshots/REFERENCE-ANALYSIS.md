## 1. Canva 분석

### 아키텍처
- **네비게이션**: 심플한 6개 메뉴 (디자인, 기능, 요금제, 비즈니스, 교육용, 도움말) + 가입/로그인
- **히어로 섹션**: 짧은 헤드라인 + 영상 배경 + 단일 CTA 버튼
- **탭 기반 콘텐츠**: HOT/AI/프레젠테이션/SNS/동영상 탭으로 기능 소개
- **인터랙티브 데모**: AI 이미지 생성 프롬프트 직접 체험 가능
- **비즈니스 섹션**: 기업용 솔루션 별도 강조
- **풍부한 푸터**: 기능/회사/요금제/아이디어/도움말/도구/커뮤니티 7개 카테고리

### 디자인 특징
- **색상**: 보라색 그라데이션 배경 (hero) + 화이트 카드 + 컬러풀한 악센트
- **타이포그래피**: 큰 볼드 헤드라인 (40px+), 여유로운 line-height
- **카드 디자인**: 큰 이미지 카드 + 오버레이 텍스트, 둥근 모서리
- **비주얼**: 실제 제품 스크린샷 + 영상 위주 (텍스트 최소화)

### SNS메이킷 적용 포인트
- **탭 기반 도구 소개** (카테고리별 탭 전환) 도입
- **인터랙티브 데모** 영역 추가 (간단한 AI 체험)
- **실제 결과물 이미지** 쇼케이스 추가

## 2. Jasper.ai 분석

### 아키텍처
- **상단 배너**: 최신 소식/리포트 안내 배너 (그래디언트)
- **히어로**: 강렬한 헤드라인 + 듀얼 CTA (Free Trial + Get A Demo)
- **소셜 프루프**: 기업 로고 무한 스크롤 마퀴
- **플랫폼 소개**: 3가지 핵심 제품 (Agents, Content Pipelines, Jasper IQ) 카드
- **Why 섹션**: 아코디언/확장형 카드 (클릭하면 상세 내용 + 이미지)
- **솔루션 역할별**: 탭으로 마케터 역할별 솔루션 (Performance/PR/Product/Brand/Content/Field)
- **고객 사례**: 벤토 그리드 레이아웃 (숫자 강조 + 인용문 + 로고)
- **트러스트**: 보안/엔터프라이즈급 인증 강조
- **리소스**: 블로그/교육/고객사례/지원/커뮤니티/지식센터 6개 카드

### 디자인 특징
- **색상**: 흰 배경 + 블루/레드/그린/옐로우 포인트 (다채로운 악센트)
- **타이포그래피**: 세리프 + 산세리프 믹스, 매우 큰 헤드라인
- **레이아웃**: 벤토 그리드 (비대칭 그리드) 활용
- **인물 이미지**: 실제 인물 사진 + 컬러 배경 조합
- **애니메이션**: Rive 기반 인터랙티브 애니메이션, Lenis 스크롤

### SNS메이킷 적용 포인트
- **기업 로고 마퀴** (신뢰도 강화) - 로고 스크롤 배너
- **벤토 그리드 고객 사례** 레이아웃
- **역할/페르소나별 솔루션 탭** (블로거/쇼핑몰/소상공인/크리에이터)

## 3. Buffer 분석

### 아키텍처
- **히어로**: 이모지 떠다니는 배경 + 이메일 입력 CTA (바로 가입)
- **실시간 숫자**: Active Users(191K+), Posts Published(7.8M+), Platforms(11)
- **핵심 기능 4가지**: Publish/Create/Community/Analyze 각각 큰 섹션
- **추가 기능**: 4개 미니 카드 (Collaborate, Mobile, Start Page, AI Assistant)
- **채널 연동**: 11개 SNS 플랫폼 아이콘 그리드
- **페르소나 탭**: Creators/Small businesses/Agencies 탭 전환
- **고객 지원**: "Human support, worldwide" 강조
- **리소스**: 5개 리소스 카드 (Free Tools, Glossary, Marketing 101 등)
- **About**: 투명 경영 지표 공개 (MAU, 매출, 팀원 수)

### 디자인 특징
- **색상**: 파스텔 컬러 (민트/라벤더/피치/옐로우) 섹션별 배경
- **타이포그래피**: 세리프 헤드라인 (우아한 느낌), 산세리프 본문
- **레이아웃**: 좌우 교대 배치 (이미지+텍스트 번갈아)
- **실제 UI 스크린샷**: 제품 대시보드 이미지 적극 활용
- **이메일 입력 CTA**: 히어로에서 바로 이메일 입력
- **투명성**: 실제 수치 공개로 신뢰 구축

### SNS메이킷 적용 포인트
- **파스텔 컬러 섹션 구분** (각 섹션별 다른 배경색)
- **핵심 기능 대형 카드** (기능별 스크린샷 + 설명 좌우 교대 배치)
- **실시간 사용자 수/생성 콘텐츠 수 카운터** (이미 있지만 더 크게)
- **페르소나별 탭** (블로거/마케터/쇼핑몰 사장님)

- **제품 영상**: 실제 제품 사용 데모 영상 임베드
- **기능 섹션**: 좌우 교대 (텍스트+이미지) 레이아웃
- **템플릿 갤러리**: 3줄 무한 스크롤 템플릿 미리보기 (자동 스크롤)
- **산업별 쇼케이스**: 탭으로 업종별 예시 영상 (Consulting/Finance/Technology 등)
- **고객 사례**: 캐러셀 슬라이더 (인용문 + 기업 로고)
- **최종 CTA**: "Join the brands" + 듀얼 CTA

### 디자인 특징
- **색상**: 보라/파랑 그라데이션 히어로 + 흰 배경 본문
- **타이포그래피**: 매우 큰 헤드라인 (가독성 최우선)
- **비주얼**: 그라데이션 배경 + 제품 스크린샷 조합
- **템플릿 마퀴**: 자동 스크롤되는 샘플 갤러리 (시각적 임팩트)
- **케이스 스터디**: 슬라이더 + 하단 로고 네비게이션

### SNS메이킷 적용 포인트
- **자동 스크롤 결과물 갤러리** (생성된 카드뉴스/상세페이지 쇼케이스)
- **고객 사례 캐러셀** (인용문 + 성과 수치)
- **업종별 활용 예시 탭** (커머스/뷰티/요식업/교육 등)

## 10. Copy.ai 분석

### 아키텍처
- **상단 배너**: 최신 리포트/이벤트 안내 배너
- **히어로**: 강렬한 헤드라인 ("The First AI-Native GTM Platform") + 이메일 입력
- **소셜 프루프**: "Trusted by 17 million users" + 기업 로고
- **유스케이스 그리드**: 6개 카드 (Prospecting/Content/Inbound/ABM/Translation/Deal)
- **고객 후기 슬라이더**: 대형 인용문 + 이름/직위
- **플랫폼 아키텍처**: 계층 구조 다이어그램 (Workflows/Actions/Agents/Tables 등)
- **블로그 캐러셀**: 최신 아티클 4개 슬라이더
- **최종 CTA**: "See the Copy.ai platform in action"

### 디자인 특징
- **색상**: 화이트 + 퍼플 악센트 + 다크 섹션 교대
- **타이포그래피**: 매우 큰 세리프 헤드라인 (인용문에)
- **구조 다이어그램**: 플랫폼 아키텍처를 시각화
- **슬라이더**: 고객 후기 + 블로그 캐러셀

### SNS메이킷 적용 포인트
- **유스케이스 카드 그리드**: 용도별 기능 소개 카드
- **대형 인용문 슬라이더**: 고객 후기를 전체 화면으로
